## Description

This project is a sample shopping list.

Purpose of this project is to show case page navigation and theme changer.

Following concepts are covered in this example:

- Create react app
- **Context**
- React Routing
- **Page Navigation**
- **Theme Changing**
- Common Components
- Custom Hooks
- Bootstrap 5 / React Bootstrap
